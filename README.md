# air-quality-analytics
Interactive Power BI dashboard for analyzing Jakarta Air Quality Index (ISPU) data using data visualization and analytics techniques.

# Overview
My first self-project using Microsoft Power BI to explore and visualize Jakarta Air Quality Index (AQI/ISPU) data. This project was created as a learning exercise to become familiar with Power BI, dashboard design, data visualization, and basic data analysis.

This dashboard visualizes historical Air Quality Index (AQI/ISPU) data collected from monitoring stations in Jakarta. It provides an interactive overview of pollutant concentrations and AQI values across different years and locations.

The dashboard includes:
- AQI (MAX) trend analysis
- AQI distribution by year
- Detailed data table
- Interactive filters (Year, Quarter, Month, Day, Station, and Critical Pollutant)

# Dataset
- Dataset: Air Quality Index in Jakarta
- Source: https://www.kaggle.com/datasets/senadu34/air-quality-index-in-jakarta-2010-2021

Features include:
- PM10
- PM2.5
- SO₂
- CO
- O₃
- NO₂
- AQI Category
- Monitoring Station
- Date
